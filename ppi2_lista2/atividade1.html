O que é uma regra CSS?

CSS é a estrutura fundamental que define como o estilo será aplicado a um ou mais elementos HTML em uma página web. Ela segue uma sintaxe específica para associar um estilo (uma ou mais declarações de propriedade e valor) a elementos selecionados

Conceito

Toda regra CSS básica é composta por duas partes principais, o Seletor(Selector) e ao bloco de declaração (Declaration Block)

  1. Seletor(Selector)
    • É o alvo da regra, ele que seleciona quais elementos HTML receberão o estilo
    • Pode ser um nome de tag HTML (EX: p, h1), uma classe ( .destaque), um ID (#cabecalho), ou uma combinação de vários.
  1. Bloco de Declaração (Declaration Block):
    • Ele contêm uma ou mais declarações de estilo para os elementos selecionados.
    • Fica sempre entre chaves ({})

                <color: red;
